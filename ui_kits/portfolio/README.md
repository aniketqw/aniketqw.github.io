# Portfolio UI Kit

High-fidelity recreation of the public portfolio at `aniketqw.github.io`. Components match the source `index.html` but are factored into reusable JSX.

**Components**
- `Nav.jsx` — fixed glass nav with gradient AS monogram + resume CTA
- `Hero.jsx` — full-viewport hero with gradient name and dual CTAs
- `AboutSection.jsx` — prose + portrait + stats
- `ProjectCard.jsx` + `ProjectsSection.jsx` — filterable grid, featured badges
- `ExperienceCard.jsx` — dark glass card with bulleted achievements
- `SkillsSection.jsx` — 3-col category cards w/ colored headings
- `ContactSection.jsx` — email/linkedin CTAs + social icon row

Open `index.html` for the interactive click-thru demo (filter projects, send mock message).
