# Admin UI Kit

Recreation of the repo manager tool (`admin.html`) that lets Aniket curate his portfolio's project list.

**Components**
- `Sidebar.jsx` — logo, username + token inputs, fetch button, live stats, legend
- `Toolbar.jsx` — title, search, export button
- `FilterRow.jsx` — All / AI-ML / Web Dev / Other / Pinned / Excluded pills
- `RepoCard.jsx` — single repo card with pin/exclude actions, chips, meta
- `OutputPanel.jsx` — JSON preview with copy button

Open `index.html` for the interactive tool — click Fetch, pin/exclude cards, watch JSON update.
