If the user invokes this skill without any other guidance, ask them what they want to build or design, ask some questions, and act as an expert designer who outputs HTML artifacts _or_ production code, depending on the need.

## Quick reference
- **Fonts:** Inter (UI/display, 300–900), JetBrains Mono (code/meta). Google Fonts CDN.
- **Tokens:** see `colors_and_type.css` — copy into any new HTML file.
- **Dark-first:** `#0a0a0a` background. All UI is dark; there is no light theme.
- **Signature gradient:** `linear-gradient(135deg, #3b82f6 0%, #8b5cf6 100%)` — use sparingly on headlines, CTA buttons, featured badges.
- **Voice:** first-person, warm, concrete ("I build AI-powered applications"). No emoji in body copy.
- **UI kits:** `ui_kits/portfolio/` (marketing site) and `ui_kits/admin/` (repo manager tool).
